알리고 API — IP 등록이 필요합니다

현재 PC의 공인 IP가 알리고에 등록되지 않아 API 호출이 차단됩니다

!

왜 IP 등록이 필요한가요?

알리고(Aligo) API는 보안을 위해 등록된 IP에서만 API 호출을 허용합니다.

현재 PC의 공인 IP가 알리고에 등록되어 있지 않으면, 토큰 생성 및 메시지 발송이 모두 차단됩니다.

오류 메시지: "토큰 생성 실패: 인증되지 않는 서버 IP로 부터의 호출 입니다."
이 오류가 나타나면 아래 절차대로 IP를 등록하세요.
1

알리고 발송 서버 IP 관리 페이지 접속

아래 링크를 클릭하여 알리고의 발송 서버 IP 관리 페이지로 이동합니다.

알리고 IP 등록 페이지 바로가기

로그인이 필요합니다. 알리고 계정으로 로그인하세요.
2

공인 IP 등록하기

아래와 같은 화면이 보입니다. 프로그램 설정에 표시된 공인 IP를 입력하고 [IP 추가하기]를 클릭하세요.

유동 IP 사용자: 4번째 칸을 비워두면 해당 대역 전체가 허용됩니다.
예: 58 . 126 . 213 . (공란) → 58.126.213.* 전체 허용
IP가 자주 바뀌는 환경이라면 이 방법이 편리합니다.
3

등록 확인

  1. 알리고에서 IP 등록 완료
  2. 프로그램의 설정 관리에서 [새로고침] 버튼 클릭
  3. 공인 IP 옆에 (등록됨)이 표시되면 성공
등록 후 반영까지 1~2분 소요될 수 있습니다. 바로 안 되면 잠시 후 다시 시도하세요.

자주 묻는 질문

Q. IP가 자꾸 바뀌는데 어떻게 하나요?

A. 4번째 칸을 비워서 대역으로 등록하세요. 또는 IP 변경 시마다 재등록이 필요합니다.


Q. 사무실과 집에서 번갈아 사용합니다.

A. 두 곳의 공인 IP를 모두 등록하면 됩니다. 여러 IP를 등록할 수 있습니다.


Q. 등록했는데 여전히 오류가 납니다.

A. IP를 정확히 입력했는지 확인하세요. VPN 사용 중이라면 VPN IP가 아닌 실제 IP를 등록해야 합니다. 반영까지 1~2분 소요될 수 있습니다.


Q. 페이지가 안 보여요.

A. 알리고 로그인 후 접속해야 합니다. 직접 URL을 입력하세요: https://smartsms.aligo.in/shop/kakaoauth.html
