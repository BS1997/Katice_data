프로그램 로그인

프로그램을 실행하면 "인증되지 않은 사용자입니다" 화면이 나옵니다. 중앙의 [로그인] 버튼을 클릭하세요.

  1. 등록된 이메일을 입력합니다
  2. [인증번호 요청]을 클릭합니다
  3. 이메일로 받은 6자리 인증번호를 입력합니다
  4. [인증 확인]을 클릭하면 로그인 완료!
인증번호는 5분간 유효합니다. 5분이 지나면 [재전송] 버튼으로 다시 받으세요.
?

등록되지 않은 이메일인 경우

아직 등록되지 않은 이메일을 입력하면 가입 신청 화면이 자동으로 나타납니다.

  1. 이메일 — 인증번호를 받을 이메일
  2. 상호(이름) — 회사명 또는 이름
  3. 휴대전화번호 — 연락 가능한 번호 (예: 01012345678)
  4. 개인정보 수집 동의 체크
  5. [가입 신청] 클릭
가입 신청 후 관리자가 확인하고 승인합니다.
승인이 완료되면 해당 이메일로 로그인할 수 있습니다.
승인 전까지는 "승인 대기 중입니다" 메시지가 표시됩니다.

또는 로그인 화면 하단의 "처음 사용하시나요? 가입 신청하기" 링크를 클릭해도 됩니다.

이미 (등록됨)으로 표시되어 있다면 이 단계는 건너뛰세요!
오류 메시지: "토큰 생성 실패: 인증되지 않는 서버 IP로 부터의 호출 입니다."
이 오류가 나타나면 아래 절차대로 IP를 등록하세요.

3-3. 등록 확인

  1. 알리고에서 IP 등록 완료
  2. 프로그램의 설정 관리에서 [새로고침] 버튼 클릭
  3. 공인 IP 옆에 (등록됨)이 표시되면 성공
등록 후 반영까지 1~2분 소요될 수 있습니다. 바로 안 되면 잠시 후 다시 시도하세요.

IP 등록 FAQ

Q. IP가 자꾸 바뀌는데 어떻게 하나요?

A. 4번째 칸을 비워서 대역으로 등록하세요. 또는 IP 변경 시마다 재등록이 필요합니다.


Q. 사무실과 집에서 번갈아 사용합니다.

A. 두 곳의 공인 IP를 모두 등록하면 됩니다. 여러 IP를 등록할 수 있습니다.


Q. 등록했는데 여전히 오류가 납니다.

A. IP를 정확히 입력했는지 확인하세요. VPN 사용 중이라면 VPN IP가 아닌 실제 IP를 등록해야 합니다.

엑셀로 대량 발송하기

수신자마다 다른 내용(매출액, 일정 등)을 보내야 할 때 엑셀을 사용합니다.

  3. 엑셀을 열어 고객 데이터를 입력합니다
  4. 하단 발송 영역에서 템플릿 선택 → 엑셀 파일 선택
  5. [테스트 발송] 후 [실제 발송]
중요: 엑셀의 메시지 내용은 승인된 템플릿과 정확히 일치해야 합니다. 변수 부분(#{고객명} 등)만 다를 수 있습니다.

템플릿 등록 가이드

T

알림톡 템플릿이란?

알림톡은 카카오에서 사전에 승인받은 템플릿만 발송할 수 있습니다. 템플릿은 일종의 메시지 양식으로, 변수(#{고객명} 등)만 바꿔가며 대량 발송이 가능합니다.

템플릿 등록은 본 프로그램에서 할 수 없습니다.
알리고 사이트에 접속해 직접 등록 → 카카오 검수(4~5영업일) → 승인 완료 후 본 프로그램에서 발송하세요.
1

알리고 사이트 접속 및 템플릿 등록

    https://smartsms.aligo.in/shop/kakaotemplate.html

FAQ

Q. 프로그램 실행 시 "인증되지 않은 사용자입니다"가 나와요.

A. [로그인] 버튼을 클릭하고 등록된 이메일로 인증하세요. 미등록 이메일이면 가입 신청을 먼저 하세요.


Q. "승인 대기 중입니다" 메시지가 나와요.

A. 가입 신청 후 관리자 승인을 기다리고 있는 상태입니다. 승인이 완료되면 로그인이 가능합니다.


Q. "토큰 생성 실패: 인증되지 않는 서버 IP" 오류가 나와요.

A. 공인 IP가 알리고에 미등록 상태입니다. 설정의 [IP 등록 방법] 버튼을 눌러 가이드를 따라하세요.


Q. "성공"이라고 나오는데 카카오톡에 안 와요.

A. 친구톡: 수신자가 채널 친구가 아닐 수 있습니다. / 알림톡: 메시지가 템플릿과 다를 수 있습니다. 알리고 관리자 페이지에서 상세 실패 사유를 확인하세요.


Q. "등록되지 않은 인증키" 오류가 나와요.

A. 설정에서 ALIGO_API_KEY와 ALIGO_SENDER_KEY 값이 뒤바뀌지 않았는지 확인하세요.


Q. 카카오비즈니스에서 발송 내역이 안 보여요.

A. 정상입니다. 알리고를 통한 발송은 카카오비즈니스 관리자센터에 표시되지 않습니다. 알리고에서 확인하세요.


Q. 한 번에 몇 명까지 보낼 수 있나요?

A. 1회 최대 500명입니다. 프로그램이 자동으로 500명씩 나누어 발송합니다.


Q. 발송 중 엑셀에 결과가 안 써져요.

A. 발송 중에 엑셀 파일이 열려 있으면 저장이 실패합니다. 발송 전에 엑셀을 닫아주세요.


Q. 알림톡 템플릿이 반려되었어요.

A. 카카오 검수 기준에 맞지 않은 경우입니다. 내용을 수정하여 새로 등록하세요. 광고성 내용은 알림톡이 아닌 친구톡으로 보내야 합니다.
